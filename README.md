# Email-Projectify
AI-powered system that transforms unstructured emails into organized, project-based knowledge.

## Overview

Email Projectify is an AI-powered web application that automatically organizes a user’s emails into structured, project-based workflows. Instead of manually sorting through inbox clutter, the system continuously analyzes incoming emails, extracts key information, and intelligently maps them to relevant user-defined projects.

## Problem

Modern inboxes act as unstructured streams of information, making it difficult to track ongoing work across multiple projects. Important updates, tasks, and context are often buried in long email threads.

## Solution

This project introduces an AI-driven pipeline that:
	•	Scrapes and processes incoming emails at regular intervals
	•	Uses large language models to summarize and extract key information
	•	Classifies emails into user-defined projects based on semantic relevance
	•	Suggests the creation of new projects when no match is found

## Goals
	•	Reduce inbox overload through intelligent automation
	•	Provide structured visibility into ongoing work
	•	Demonstrate real-world application of AI in productivity systems
